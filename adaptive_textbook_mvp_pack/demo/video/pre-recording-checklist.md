# 錄影前檢查清單

## 前端畫面

- [ ] 第一眼是中文高中課本，不是 dashboard。
- [ ] 畫面中沒有「demo」「prototype」「錄影」等破壞情境的文字。
- [ ] 共筆標記位於「一、從海底擴張到板塊構造」段落句點後。
- [ ] 共筆 hover 彈窗文字不會溢出。
- [ ] 學生答錯 B 後有清楚的正確答案與解析。
- [ ] 答錯後可以跳到對應觀念。
- [ ] 跳到對應觀念後，AI 會自動畫螢光重點並插入個人筆記。
- [ ] 個人筆記區有 `新增到共筆` 按鈕。
- [ ] 加入共筆後，下一位學生 hover 共筆可看到心得、AI 整理與常見錯誤。

## 錄影流程

- [ ] 開啟 `http://127.0.0.1:3000`。
- [ ] 確認 `npm run record:demo` 可以啟動。
- [ ] 影片輸出到 `output/demo-video/adaptive-textbook-demo.webm`。
- [ ] 4 張 keyframe 截圖都有產生。
- [ ] 影片中滑鼠移動不會太快。
- [ ] 每個 AI 生成結果至少停留 2 秒。
